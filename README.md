# Next-Gen-Store

## Setup and Install
- clone project: https://github.com/NextGenStore/Next-Gen-Store.git
- composer install
- cp .env.example .env
- php artisan key:generate

## Docker Setup
- run: docker compose up -d
- check env with database and service
## Run Laravel 
- php artisan serve
- open port: 127.0.0.1:8000
